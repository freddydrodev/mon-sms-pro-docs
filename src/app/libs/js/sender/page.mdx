## Introduction

Le nom de l'expéditeur **(senderId)** est le nom utilisé pour identifier l'expéditeur du SMS.
Il peut représenter votre marque, votre entreprise, etc...
Il utilise uniquement des caractères alphanumériques et est limité à 11 caractères maximum.

## Utilisation

Après avoir [Initialisé](/libs/js/get-started/#init) votre bibliothèque.
vous pouvez utiliser la propriété `sender` de votre variable.

```ts filename="/api/sender" showLineNumbers {1,5-8,18,28-30}
import { sms } from "@/libs/sms";

const createSender = async () => {
  try {
    const res = await sms.sender.create({
      name: "My Sender",
      description: "This is the name of my marque",
    });

    console.log(res);
  } catch (error) {
    console.error(error);
  }
};

const getSenders = async () => {
  try {
    const res = await sms.sender.list();

    console.log(res);
  } catch (error) {
    console.error(error);
  }
};

const getSenderCampain = async () => {
  try {
    const res = await sms.sender.campains({
      id: "id_of_the_sender",
    });

    console.log(res);
  } catch (error) {
    console.error(error);
  }
};
```

Comme vous pouvez le constater nous avons utilisé la variable `sms` créée
précédemment pour envoyer et vérifier le code d'authentification à travers
3 fonctions distinctes représentant ici 3 API.
