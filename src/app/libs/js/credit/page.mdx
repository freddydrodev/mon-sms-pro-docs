## Introduction

Les crédits représentent le solde disponible sur votre compte pour envoyer des SMS. Chaque message envoyé déduit un certain montant de votre solde en fonction des tarifs appliqués. Il est important de surveiller votre balance de crédits pour vous assurer que votre compte dispose de suffisamment de fonds pour les opérations prévues.

<Callout type="info" emoji="ℹ️">
  Assurez-vous de recharger votre compte avant que votre solde ne soit épuisé
  pour éviter toute interruption de service. 1 crédit vaut en moyen $0.04 soit
  25F CFA. Les Message Flash utilise plus de crédits que les SMS.
</Callout>

## Utilisation

Après avoir [Initialisé](/libs/js/get-started/#init) votre bibliothèque.
vous pouvez utiliser la propriété `user` de votre variable.

```ts filename="/api/credit" showLineNumbers {1,5}
import { sms } from "@/libs/sms";

const getCredit = async () => {
  try {
    const res = await sms.user.credit();

    console.log(res);
  } catch (error) {
    console.error(error);
  }
};
```

## Propriétés

### sms.user.credit

Cette méthode ne recoit aucun argument.
